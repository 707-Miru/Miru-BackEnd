1. 우선순위 A 규칙: 필수 (에러 방지)

   - 컴포넌트 이름에 합성어 사용

     "모든 HTML 엘리먼트의 이름은 한 단어이기 때문에 합성어를 사용하는 것은 기존 그리고 향후 HTML 엘리먼트와의 충돌을 방지해줍니다."

   ```
   ###Bad!
   
   Vue.component('todo', {
   // ...
   })
   
   ​``` js
   export default {
   	name: 'Todo',
   	// ...
   }
   ```

   ```
   ###Good!
   
   Vue.component('todo-item', {
   // ...
   })
   
   ​``` js
   export default {
   	name: 'TodoItem',
   	// ...
   }
   ```
